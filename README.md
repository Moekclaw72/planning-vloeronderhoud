# Planning Vloeronderhoud 2026

Realtime planning tool voor vloeronderhoud met Firebase sync.

## Features

- Realtime samenwerking (4+ gebruikers)
- Status tracking (Bezig / Klaar) met naam en tijdstempel
- Live activiteit banner — zie wie waar bezig is
- Push notificaties bij statuswijzigingen
- Delen via WhatsApp, Mail of Kopieer
- Opmerkingen per week
- Uitloop / vertraging beheer
- Kleur-coded locaties met legende
- PWA — installeerbaar als app op telefoon
- Volledig offline capable met Firebase sync

Open `index.html` in een browser. Vul je naam in en begin met plannen.
